软件创新中心介绍

软件创新中心于2014年10月由华东交通大学软件学院筹备建立，由占东明讲师负责带领，主要内容由学生团队进行日常管理以及运营，其核心为“以软件技术为基础，提高学生综合素养”，中心团队规模在20人左右。目前团队已经建立较为全面、初步有效的制度，能够为学生团队的组织架构、日常运营等活动提供了制度保证，并摸索到了以项目为驱动而展开的学生日常内容，养成了较为良好的技术氛围和人文氛围。

历届团队概况

第一届以优培网项目发起，并建立学生团队，以刘佳诚等同学为核心团队，对优培网项目进行需求调研、开发，并成功上线。第二届以黄松鹏等同学为核心团队，对学生团队的可持续建设，中心日常内容进行积极探索。第三届以陈希杨等同学为核心团队，对学生团队建立初步的制度，以及对学生团队日常内容做出了初步的实践。

软件分流方向

前端开发

前端对于网站来说，通常是指，网站的前台部分包括网站的表现层和结构层。因此前端技术一般分为前端设计和前端开发，前端设计一般可以理解为网站的视觉设计，前端开发则是网站的前台代码实现，包括基本的HTML和CSS以及JavaScript/ajax，现在最新的高级版本HTML5、CSS3，以及SVG等。html是内容层，它的目的是表示一个HTML标签在页面里是个什么角色；CSS是样式层，它的目的是表示一块内容以什么样的样式（字体、大小、颜色、宽高等）显示；JS是行为层，它要做的是当用户触发某些行为时，会给内容和样式带来什么样的改变。

后端开发

后端涉及多个方面。开发语言：包括但不限于（php, python, java, scala, go, ruby) 之一。，能够使用一门或者数门语言构造大型应用。运维：了解ansible, salt,puppet等自动化运维技术， 了解docker， 熟练掌握shell脚本处理线上问题。数据库：知道常见的数据库的优化以及运维，能够分析sql并且调优使之满足性能。大数据：了解 （hadoop/storn/spark）技术栈， 能够使用大数据技术栈解决常见的大数据问题。机器学习：知道常见的机器学习算法，能够使用机器学习和深度学习解决一些人工智能问题，比如聊天机器人， 反spam， 机器推荐等。简而言之，你需要什么，我就给你什么。

UI设计

UI设计主要涉及三个方面：研究界面—-图形设计师 Graphic UI designer，简称 GUI,国内目前大部分 UI 工作者都是从事这个行业。包括（网页设计，软件界面，移动端界面设计），每天工作做着各种界面设计。交互设计师，简称IXD,主要的工作内容是绘制原型图！用户体验研究师，简称UE.主要是通过各种方法去了解用户现在需要什么样的体验什么样的界面.

移动开发

移动开发主要是针对安卓和iOS的开发，关于安卓，了解一些安卓编程的基础，学习安卓入门知识，比如简单的数据存储、安卓界面布局、网络编程，学习安卓四大组件等等。初步了解一个简单APP的生成过程。学会了安卓所需基础知识后，就可以开始尝试做一些小的APP试手。当经验积累成熟的时候，就可以着手做一些比较大的APP项目。

网络安全

网络安全是指网络系统的硬件、软件及其系统中的数据受到保护，不因偶然的或者恶意的原因而遭受到破坏、更改、泄露，系统连续可靠正常地运行，网络服务不中断。关于安全技术，知道各种安全攻击方式（xss, csrf, sql注入） 代码实现能够规避常见的安全漏洞, 处理各种网络攻击事件。

产品经理

IPM（Internet Product Manager），互联网产品经理是互联网公司中的一种职能，需理解业务需求、感受用户需求、并转化为产品功能需求，而后和技术团队、设计团队、销售团队等进行沟通，进一步明确产品功能需求、设计并完成软件开发。而后负责互联网产品的推广以及生命周期的演化。互联网产品经理在互联网公司中处于产品的核心位置，需要非常强的沟通能力、协调能力、市场洞察力和商业敏感度。不但要了解消费者，了解市场，还要能跟各种风格迥异的团队，如开发团队和销售团队进行默契的配合。
